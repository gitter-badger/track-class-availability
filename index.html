Welcome to Drexel Class Track Availability

Track Class Availability (trackClassAvailability.py) is a tool written in Python (2.7) that allows a user to track numerous classes through WebTMS. The user can choose whether to be notified when a section opens or closes. Track Class Availability will then check all your classes via WebTMS every hour to see if the sections you are tracking open or close. If the classes you are tracking open or close, you will be notified of the status change via email.

Make sure you have Python installed. Then just click the "Download as Zip" above and install BeautifulSoup before using!. When those are installed, simple open a Terminal and cd into the directory. For example if my unzipped folder is in my downloads, I would open the Terminal and type cd ~/Downloads/heinst-track-class-availability-7771aa3/ and then type python python trackClassAvailability.py and follow the setup process. If you want to add more classes, follow the exact format of line that program created for you in the file ClassesToTrack.txt. You can run this on a cron job so that way it will check every set interval for class status changes and email you.
